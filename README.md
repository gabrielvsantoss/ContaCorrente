# ContaCorrente

- Neste trabalho da Academia do Programador realizei a construção de uma conta corrente que possui algumas funcionalidades especificas.

## Video do projeto:

![](https://i.imgur.com/OcJgNXc.gif)

## Funcionalidades:

**Saque**: Permite sacar dinheiro da conta.
**Depósito** Permite depositar dinheiro na conta;
**Visualizar Extrato** Permite visualizar operações realizadas
**Depósito entre contas**  Permite transferir para uma conta.
